Divvycam targets iOS devices and makes it easier to share photos between devices. The application was developed using Titanium which heavily relied on Javascript code. Some key concepts of the code include photo compression, in-app purchase interactions, and API requests. An API was also built with a custom load-balancer and web-node solution on the Amazon Web Services platform. Essentially, servers would spin up and down according to load percentages. The splash page of Divvycam can be found here: http://divvy.burst-dev.com While you can grab the app at: http://itunes.apple.com/us/app/divvycam/id480270517?ls=1&mt=8
