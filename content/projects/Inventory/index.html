Inventory is a open-source project initially developed for a high-school robotics team. It catalogs all the available items and tracks all items ordered from vendors. Although I haven't worked on this project for quite a long time, you can access a copy of it at: https://github.com/sjlu/Inventory-Cataloger You can also view a copy here: http://styx.stevenlu.com/~sjlu/Inventory-Cataloger/
